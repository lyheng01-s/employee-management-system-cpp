# Employee Management System

This project is a console-based Employee Information Management System developed using C++.

The system stores employee information using CSV/Excel-style files and allows administrators and employees to manage information through a simple console interface.

## Features

### Admin Features
- Add new employee records
- View employee information
- Update employee records
- Delete employee records
- Search employee information

### Employee Features
- View personal information
- Update selected personal details

## Tech Stack

- C++
- File Handling
- CSV/Excel-style Data Storage
